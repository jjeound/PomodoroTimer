# 📚 Custome Pomodoro Timer

> 색상, 폰트, 크기, 배경 등 다양하게 커스텀할 수 있는 뽀모도로 타이머 앱

<a href="https://play.google.com/store/apps/details?id=com.jje.timer">다운로드</a>

---

## 🧩 프로젝트 소개

- **개발 목적**
  
  - 포모도로 타이머는 도서관이나 독서실처럼 개방된 공간에서 여러 사람의 시선이 닿는 환경에서 자주 사용됩니다. 공부를 하면서 휴대폰 사용하는 것을 방지하고 생산성을 높이기 위해서인데, 그렇기 때문에 타이머 앱의 UI는 하나의 오브제처럼 다뤄질 수 있다고 생각하였습니다. 따라서 다른 앱보다 UI가 더 이쁘고 세련되어 있다면, 그것만으로도 충분한 경쟁력을 갖출 수 있다고 생각하여 이 앱을 만들게 되었습니다.
  - 저는 전문적인 디자인 교육을 받은 적은 없지만, 다양한 레퍼런스를 참고하며 미니멀하고 감각적인 디자인을 구현하기 위해 노력했습니다. 또한 부족한 디자인 실력을 보완하기 위해 사용자 커스터마이징 기능을 강화했습니다. 특히 학생 사용자층을 고려해, 배경 화면에 사용자가 원하는 아이돌 사진이나 이미지를 넣어 자신만의 타이머를 만들 수 있도록 했습니다.

| UI | Jetpack Compose |

| Database | Room / DataStore |

## 📱 주요 기능

- UI 커스텀: 타이머의 전경, 초침, 엣지 색상, 배경 색상과 이미지, 숫자 폰트와 크기 커스텀 기능
- 타이머 커스텀: 타이머의 간격, 휴식 시간, 반복 횟수, 시작과 만료시에 효과음 커스텀 기능
- 배경 이펙트: Lottie를 활용한 눈이 내리는 배경 이펙트 기능

## 📈 결과 및 배운 점

- `Room DB`와 `DataStore`를 사용하여 데이터를 저장하고 활용하는 방법을 알게 되었다.
- `Room DB`에서 타입 컨버터를 활용하는 방법과 마이그레이션을 사용하는 방법을 알게 되었다.
- `Stable`과 `Immutable` 어노테이션과 `State hoisting`을 사용하여 불필요한 `Recomposition`을 최소화하여 성능을 최적화하는 방법을 알게되었다.
- `Lottie`**의** 사용하여 복잡한 애니메이션을 쉽게 구현하는 법을 알게되었다.
- `MediaPlayer`를 사용하여 사운드 효과를 구현하는 법을 알게되었다.
